Tips y Trucos

  • Habilitar la virtualización en BIOS: Asegúrate de que la opción de virtualización esté activada en tu BIOS para mejorar el rendimiento.
  • Usar VDI para discos dinámicos: Elige un disco VDI (VirtualBox Disk Image) para que el tamaño del disco virtual aumente solo cuando sea necesario.
  • Instalar Guest Additions: Esto agrega soporte para gráficos mejorados, el portapapeles compartido y carpetas compartidas entre el sistema huésped y la máquina virtual.
