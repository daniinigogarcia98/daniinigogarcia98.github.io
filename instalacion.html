Instalación de Windows en VirtualBox

Para instalar Windows en VirtualBox, sigue estos pasos:

  1. Descargar e instalar VirtualBox desde el sitio oficial.
  2. Crear una nueva máquina virtual y seleccionar Windows como sistema operativo.
  3. Asignar memoria RAM y configurar el disco duro virtual.
  4. Iniciar la máquina virtual e insertar el archivo ISO de Windows.
  5. Completar la instalación de Windows como lo harías normalmente en una PC física.
